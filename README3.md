# PROJECT_3 : Password Generator

## Project Description
This is a simple Python Password Generator that creates secure random passwords using letters, numbers, and special characters.

## Features
- Generates random passwords.
- Allows user to choose password length.
- Validates user input.
- Automatically sets password length to 12 if:
  - Length is less than 8.
  - Invalid input is entered.
- Uses uppercase letters, lowercase letters, digits, and special symbols.

## Modules Used
- random
- string

## How to Run
1. Save the file as `python project3.py`.
2. Open Command Prompt or Terminal.
3. Navigate to the project folder.
4. Run:

```bash
python "python project3.py"
```

This Password Generator can be used in:
